# 🚀 OL Calculator - Деплой-инструкция

## Структура файлов
- `index.html` - Основной интерфейс калькулятора
- `admin.html` - Панель продукт-менеджера (редактор конфига + история)
- `config.js` - База контента, правила маппинга, кейсы
- `styles.css` - Стили, анимации, адаптив
- `app.js` - Логика валидации, расчёта, рендера КП, экспорта

## Быстрый запуск
1. Создайте папку, положите все 5 файлов внутрь
2. Откройте `index.html` в браузере (Chrome/Safari/Edge)
3. При первом входе введите пароль: `Partner2026!`
4. Для хостинга загрузите папку на GitHub Pages, Vercel, Netlify или внутренний Nginx.

## Как менять правила и контент
1. Перейдите в `admin.html`
2. Отредактируйте JSON в поле редактора
3. Нажмите `✅ Проверить JSON` → `💾 Скачать config.js`
4. Замените `config.js` на хостинге новым файлом (кэш сбросится автоматически при обновлении страницы)

## Особенности реализации
- 🎨 Анимации появления карточек с каскадной задержкой
- 💬 CSS-тултипы при наведении на название инструмента
- 📜 История генераций сохраняется в `localStorage` браузера менеджера
- 🗑 Кнопка «Сбросить историю» доступна только в `admin.html`
- 🔐 Клиентская авторизация (для внутренних инструментов достаточно)
- 📦 Экспорт: копирование в буфер, PDF (html2pdf), `mailto:`
